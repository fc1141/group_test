# 예상매출액 산정서 자동화 시스템

가맹본부 담당자를 위한 법적 기준 준수 매출 산정 도구

## 기능
- A형/B형 매출 산출 방식 지원
- 법적 기준(1.7배 비율) 자동 검증
- PDF/Markdown 리포트 자동 생성
- 상권 분석 및 경쟁점 현황 관리

## 설치 및 실행

```bash
pip install -r requirements.txt
streamlit run main.py
```

## 사용법
1. 기본 정보 입력 (가맹본부, 가맹희망자)
2. 상권 분석 데이터 입력
3. 매출 산출 방식 선택 및 계산
4. 결과 확인 및 리포트 다운로드

## 법적 근거
- 가맹사업법 시행령 제9조 제3항, 제4항
- 최고/최저 매출액 1.7배 비율 준수
